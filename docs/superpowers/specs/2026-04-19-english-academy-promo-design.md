# 영어학원 홍보 정적 웹사이트 설계서

- 작성일: 2026-04-19
- 범위: 데모용 1페이지 정적 랜딩 사이트
- 대상: 초/중/고 학생 및 학부모
- 핵심 CTA: 카카오톡 오픈채팅 문의
- 기술: 순수 HTML/CSS 단일 파일(`index.html`)

## 1. 목표

초/중/고 학생 및 학부모가 학원의 신뢰성과 성과를 빠르게 이해하고, 카카오톡 오픈채팅으로 문의를 남기도록 유도한다.

## 2. 정보 구조(IA)

페이지 섹션 순서:
1. 히어로(핵심 메시지 + CTA)
2. 학원 강점 3가지
3. 성적 향상 사례
4. 학부모 후기
5. 수업/관리 방식
6. 프로그램 안내(초/중/고)
7. 위치/오시는 길
8. FAQ
9. 하단 CTA

## 3. 디자인 방향

- 톤앤매너: 프리미엄 + 성과 중심
- 메시지 방향: 학부모 안심형 헤드라인
- 시각 원칙:
  - 여백 중심의 정돈된 레이아웃
  - 고대비 타이포그래피
  - 카드 UI로 사례/후기 가독성 강화
  - CTA는 시선이 머무는 지점마다 반복 배치

## 4. 콘텐츠 설계

### 4.1 히어로
- 헤드라인: 학부모 안심형 메시지
- 서브카피: 관리 체계 + 성과 근거 요약
- 버튼: 카카오톡 오픈채팅 문의

### 4.2 학원 강점 3가지
1) 내신/수능 성적 향상 실적
2) 소수정예·개별 맞춤 관리
3) 초중고 연계 커리큘럼

### 4.3 성적 향상 사례
- 카드형 3~6개
- 필드: 학생 유형, 학습 기간, 성과 변화

### 4.4 학부모 후기
- 카드형 3개 이상
- 요약형 문장으로 신뢰도 강화

### 4.5 수업/관리 방식
- 단계형 설명: 진단 → 학습설계 → 수업 → 점검 → 피드백

### 4.6 프로그램 안내(초/중/고)
- 3열 카드 또는 비교형 구성
- 각 트랙별 목표/운영 방식 제시

### 4.7 위치/오시는 길
- 주소, 교통 요약, 운영 시간(데모 데이터)

### 4.8 FAQ
- 등록 절차, 반편성, 보강, 상담, 과제관리

## 5. 상호작용/전환 설계

- 상단 네비게이션 앵커 이동
- 주요 섹션마다 CTA 버튼 반복
- 하단 고정 문의 버튼 제공
- 모든 CTA는 동일한 오픈채팅 링크를 참조

## 6. 기술 설계

- 파일: `index.html` 단일 파일
- 스타일: `<style>` 내장 CSS
- 스크립트: 불필요한 JS는 사용하지 않음(정적 구조 우선)
- 링크 변수화: 오픈채팅 URL 교체 포인트를 한곳에 모아 관리

## 7. 반응형/접근성

- 모바일 우선 레이아웃
- 터치 타깃 충분한 버튼 크기
- 텍스트 대비 확보
- 제목 계층(h1~h3) 일관성 유지

## 8. 검증 기준

1. 모바일/데스크톱에서 레이아웃 깨짐이 없다.
2. 모든 CTA 버튼이 정상 클릭된다.
3. 상단 앵커 네비게이션이 정상 이동한다.
4. 첫 화면에서 안심/성과 메시지가 분명히 전달된다.

## 9. 비범위

- 백엔드 연동
- 실시간 상담 폼
- 관리자 페이지
- 다국어 지원

## 10. 구현 메모(데모용)

- 학원명, 주소, 전화번호, 오픈채팅 URL은 데모 텍스트로 우선 구성한다.
- 실제 운영 전 사용자 제공 정보로 치환한다.
